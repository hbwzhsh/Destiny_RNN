Destiny Recurrent Neural Network

About the Project


Hello everyone! I am an avid Destiny Player and Software Engineer by trade. I got the idea for this project from a post in /r/hearthstone, where a user took all of the cards currently available in Hearthstone and put them through a neural network. After training the network, the user then had the network generate cards. I found the results facinating!

I decided that I wanted to try this with Destiny Guns! So I began on my Python journey In order to train the network, I wanted to give it a good representation of what a gun should be like in Destiny. To do this, I downloaded the Destiny Manifest, which is a large SQL table that contains information about every thing in the game. It is what fuels the game and all companion applications! After downloading this manifest, I wrote a Python script to navigate through the manifest and output every possible roll of every gun in the game. I amassed 100GB of text files containing all of these rolls!

After creating all of the training data, I created the neural network. I decided to use a recurrent neural network, with 2 GRU layers. This particular type of neural network is good for relating previously created output with the current state. In other words, this network type has a memory of what it has seen and what it has made, and relates the two. Using Python with the help of the library Theano, which utilized the GPU of computer, I created a network that will learn from all of my training data. [Add some statistics from the training]
